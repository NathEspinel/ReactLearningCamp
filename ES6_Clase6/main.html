Ejercicios Clase 6

Tema: ECMA 6

Ejercicios a desarrollar:

  1. Declarar una variable empleando const con el valor de tu nombre y otra utilizando let con el valor de tu edad. Imprimir ambos valores en consola.
  2. Utilizar un template de string para crear una cadena que incluya tu nombre y edad de manera legible.
  3. Crear una función que tome dos números como parámetros y devuelva el resultado de sumarlos, empleando una arrow function para definir la función
  4. Crear un array de números y utilizar el método map para duplicar cada número del array. Luego, imprimir el nuevo array por consola.
  5. Utilizar el método filter para filtrar los números pares del array creado en el punto anterior. Imprimir el nuevo array por consola.
  6. Crea una función que tome un array de números como parámetro y devuelva la suma de todos ellos. Utiliza el método reduce para implementar la función.
  7. Utiliza el operador spread para combinar dos arrays en uno solo.
  8. Crea un objeto con información sobre una pelicula, incluyendo título, género y año de estreno. Utiliza el destructuring para extraer esta información en variables separadas.
  9. Crea una función que tome un objeto como parámetro y devuelva una cadena con información legible sobre la película. Utiliza templates de strings y destructuring para obtener los valores del objeto.
  10. Crea una clase Persona con propiedades nombre y edad. Luego, crea una clase Estudiante que herede de Persona y agregue una propiedad curso. Crea un objeto Estudiante con información ficticia e imprime su nombre, edad y curso por consola.
  11. Crea un módulo que exporte una función que tome un número como parámetro y devuelva una promesa que se resuelva con el resultado de elevar el número al cuadrado.
  12. Utiliza async/await para llamar a la función del módulo creado en el ejercicio anterior y obtener el resultado. Imprime el resultado por consola.
  13. Crea una función que tome un número como parámetro y simule un error si el número es negativo. Utiliza try-catch para manejar el error e imprimir un mensaje de error por consola.
  14. Utiliza LocalStorage o SessionStorage para almacenar el resultado obtenido en el ejercicio 12 y mostrarlo en la página web.
  15. Combina todos los conceptos aprendidos en los ejercicios anteriores para crear una pequeña aplicación que permita al usuario ingresar su nombre y edad en un formulario. Al enviar el formulario, la aplicación debe mostrar un mensaje de bienvenida con el nombre y edad ingresados y guardar estos datos en LocalStorage para que se conserven al recargar la página.
